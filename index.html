Pickle

Pickle

Pickle is a secure backend generator for Go services. It turns schema and access intent into repository-owned code: request binding, typed query builders, RBAC-aware handlers, GraphQL exposure controls, and generated metadata that security tooling can inspect.

Its Squeeze checks make the generator self-auditing. Instead of relying on review memory, Pickle carries security expectations into generated code and gives the repository repeatable checks for drift.
